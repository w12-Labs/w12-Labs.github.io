Frequent Q/A

  • Q1: Where can we download the datasets and code?
    • Both recorded and live sensor data, as well as the necessary code will be made available during the tutorial.

  • Q2: Is a GPU or high performance cloud support required?
    • Although a GPU-based computer will be available during the tutorial, a CPU-only (i5) configuration performed 7-14 fps for non-image based tasks, and 1fps for image-based tasks. For the purposes of this tutorial, a CPU-only setup should be fine.

  • Q3: How do I know if my computer is correctly setup for this tutorial?
    • You should be able to run (CPU only or w/GPU) the OpenAI Gym PyTorch tutorial at the link here with no errors.

  • Q4: Can I use TensorFlow instead of Pytorch in this tutorial?
    • Yes - you can use TensorFlow, or any other framework you wish, as long as you are proficient in it. The instructions will be provided in Pytorch only - so you'll need to translate to your own framework. Deployment on the rover will be carried out using Pytorch only.
